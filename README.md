# Krypton OS 
Krypton OS is an incredibly lightweight unix-based web OS

# Features 
- Full x86_64 support        
- Assembly support       
- syscalls     
- A terminal    
- The ability to load and create apps, with 4 major types: Javascript, assembly/x64, Html, and Proxed web apps (similar to electron)
